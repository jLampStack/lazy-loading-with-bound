Requirements

You pretty much just need a Code Editor, Dev Server and Bound.js

For this project I'm using the server that comes with Parcel web application bundler to serve the project

Getting Started

  • npm init -y   -- Intializes project and creates package.json

  • npm i -g parcel-bundler   -- Installs Parcel globally and allows us to use its dev server

  • npm i bounds.js --save   --Saves Bound as a dependency

  • Add project files   -- index.html, index.js and a css style stylesheet

  • Import Bound at top on line 1 of index.js   -- import Bound from 'bounds.js';

  • Serve project from root dir   -- parcel index.html
